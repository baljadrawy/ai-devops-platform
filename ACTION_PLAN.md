**المهام:**

**الملفات المتأثرة:**
- `backend/server.js`
- `backend/plugins/FileSystemPlugin.js`
- `package.json`
- `.env.example`

**الاختبارات:**
```bash
# اختبار 1: بدون API Key
curl -X POST http://localhost:5000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"test"}'
# يجب أن يرفع مع 401

# اختبار 2: Shell Injection
curl -X POST http://localhost:5000/api/filesystem/execute \
  -H "X-API-Key: $API_KEY" \
  -d '{"command":"ls; rm -rf /"}'
# يجب أن يرفع مع error message
```
